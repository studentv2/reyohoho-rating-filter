# reyohoho-rating-filter
Этот userscript внедряет компактную, адаптивную панель фильтрации карточек фильмов и сериалов по рейтингу в интерфейс ReYohoho. Панель интегрируется в существующие элементы управления (например, `.controls`, `.filter-card`, `time-card` и т.п.), не использует внешние библиотеки и трекеры, автоматически подстраивается под тему сайта и устойчива к SPA-навигации. Интерфейс позволяет выбрать источник рейтинга (встроенный рейтинг ReYohoho, «Кинопоиск» или IMDb) и задать желаемое значение — по этому порогу производится динамическая фильтрация карточек на странице.


Использование:

1) Перейдите на страницу списка (/lists или /lists/) либо на /top.

2) В блоке управления (рядом с другими фильтрами) появится кнопка Фильтр. Нажмите её для развёртывания панели или сворачивания.

3) Выберите платформу рейтинга (ReYohoho / КП / IMDb).

4) Введите значение рейтинга в поле Рейтинг. Примеры вводов: 8, 8.1, 9,0.

5) Нажмите Применить — на странице останутся только карточки с совпадающим рейтингом. Краткая статистика показывает, сколько карточек отображено.

6) Нажмите Сброс для отмены фильтра и показа всех карточек.


Особенности ввода:

-Если введено целое число (например, 8), сравнение производится по округлённому значению рейтинга карточки.

-Если введено число с дробной частью (например, 8.1), сравнение учитывает дробную часть с допуском ±0.05.
